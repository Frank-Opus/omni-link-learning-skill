# Entrepreneur Guide - Omni Link Learning

## 为什么创业者需要这个技能？

作为 AI 创业者，你每天面临：
- 📊 **海量信息**: 行业报告、竞品动态、技术进展、投资人观点
- ⏰ **时间稀缺**: 要产品、要市场、要融资、要团队
- 🎯 **决策压力**: 每个选择都影响公司生死

**Omni Link Learning 帮你**:
1. **30 秒抓住核心** - 不用看完 1 小时视频
2. **提取战略洞察** - 不只是总结，而是 actionable insights
3. **转化为行动** - 直接生成创业者行动清单
4. **团队知识共享** - 生成学习计划，快速拉齐团队认知

## 最佳实践

### 1. 选择高质量内容
**优先**:
- ✅ 官方渠道 (红杉/YC/知名投资人)
- ✅ 一手信息 (创始人访谈/财报电话会)
- ✅ 深度内容 (30min+ 视频/长文)
- ✅ 数据驱动 (有具体数字/案例)

**避免**:
- ❌ 二手解读 (营销号/标题党)
- ❌ 纯观点无数据
- ❌ 过旧内容 (>1 年，除非经典)

---

### 2. 使用 ASR 获得完整转写
**对于视频/播客**:
```bash
python3 scripts/fetch_source.py \
  --input "https://..." \
  --outdir ./omni_learning_output \
  --asr-fallback \
  --asr-model large-v3-turbo \
  --asr-language zh
```

**为什么**:
- 字幕 often 不完整 (尤其专业术语)
- ASR 提供 100% 转写
- 便于后续搜索/引用

**GPU 加速**:
- 有 NVIDIA GPU: 5-10 分钟 (1.5h 视频)
- 无 GPU: 30-60 分钟
- 检查：`nvidia-smi`

---

### 3. 关注 Section C (战略启示)
**这是为你定制的部分**:
- 竞争格局分析
- 市场机会识别
- 行动清单 (本周/本月/本季度)

**不要只读 Section A/B** - 那是通用总结
**重点看 Section C** - 那是你的战略武器

---

### 4. 建立知识管理系统
**建议结构**:
```
~/startup-knowledge/
├── competitors/       # 竞品分析
├── market-research/   # 行业研究
├── technology/        # 技术学习
├── investors/         # 投资人观点
├── policy/            # 政策解读
└── action-items/      # 行动清单追踪
```

**每次分析后**:
1. 保存输出到对应目录
2. 更新行动清单追踪表
3. 同步给团队 (Notion/飞书)

---

### 5. 团队学习流程
**周会学习环节** (30 分钟):

**会前** (创始人):
- 用 omni-link-learning 分析本周重要内容
- 生成学习计划和行动清单

**会中** (团队):
- 5 分钟：分享核心摘要 (Section A)
- 10 分钟：讨论战略启示 (Section C)
- 10 分钟：分配行动项
- 5 分钟：Q&A

**会后**:
- 行动项进入 Jira/飞书任务
- 下周追踪进度

## 常见问题

### Q: ASR 转写质量差怎么办？
**A**: 
1. 检查 CJK 比例 (<0.05 说明质量差)
2. 重新运行 with `--asr-model large-v3` (非 turbo)
3. 检查音频质量 (背景音乐/多人对话影响识别)
4. 手动校对关键段落

### Q: 分析太泛用怎么办？
**A**:
1. 在输入时提供背景："[你的行业] + [视频 URL]"
2. 分析后追问："这对 [你的具体业务] 意味着什么？"
3. 使用 mckinsey-consultant 做深度定制分析

### Q: 如何追踪行动项执行？
**A**:
1. 用 planning-with-files 创建 task_plan.md
2. 每周回顾 action_items.md
3. 团队会同步进度
4. 完成项归档到 knowledge base

### Q: 团队如何访问这些分析？
**A**:
1. 保存到共享云盘 (Notion/飞书文档)
2. 设置分类标签 (#竞品 #行业 #技术)
3. 新成员 onboarding 时阅读关键分析
4. 定期 (月度) 回顾更新

## 高级技巧

### 技巧 1: 批量分析
**场景**: 快速了解一个主题
```bash
# 分析 5 个竞品访谈视频
for url in url1 url2 url3 url4 url5; do
  python3 scripts/fetch_source.py --input $url --outdir ./output_$url
done

# 然后用 LLM 综合对比分析
```

**场景**: 监控行业趋势
- 每月分析 3-5 个投资人观点

**场景**: 融资准备
- 分析目标投资人过往观点
- 提取他们的评估标准
