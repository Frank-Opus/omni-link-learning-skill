## 🎯 核心摘要（30 秒速读）

这是一期**学校学不到的实战干货**，专门教新人小白如何快速入门**高底薪的大客户销售**。主讲人用自己的成功经验，从零开始拆解：

**核心目标：**
- 快速开第一单，度过试用期
- 3 个月内必须开单，保住薪资
- 维护客户关系，积累人脉资源
- 最终实现白手起家、财务自由

**关键方法：**
- 陌生拜访：直接冲工厂/门店，一人一包烟一瓶红牛
- 一找二策略：邀请客户考察第三家工厂，一来二去建立关系
- 七步维护法：从市场动态到资源争取的全流程
- 需求挖掘：创造需求、验证需求、识别虚假需求

**适合人群：** 销售新人、想转行做大客户销售、想提高底薪的 BD/客户经理

## 📝 关键要点（15 点）

### 1️⃣ 第一单比什么都重要
> "对于新人来说，你去开第一单比什么都重要。因为你首先你得先去证明，你是一个可以卖货的人。"

**核心逻辑：**
- 成交第一单 = 保住薪资 = 度过试用期
- 只有开过单，领导才不会给你太大压力
- 第一单完成不了，后面都是扯淡

**行动建议：** 三个月内必须开出第一单，这是保命单。

### 2️⃣ 找师傅带 + 陌拜客户
> "如果你作为一个新人，你到里面一定会有一个大师傅带。三个月内一定要去开出第一单，尽可能去实现一个客户的陌拜。"

**方法：**
- 找师傅学技巧
- 自己多陌拜积累客户资源
- 搭配师傅技巧完成单子，拿到提成

### 3️⃣ 2B 大客户销售精髓：拜访客户
> "做大客户销售这个行业里面，最重要的就是去拜访客户。怎么去跟他熟络，怎么去跟客户聊天，才是渗入行业最重要的问题。"

**关键：** 不是技战术，不是学习能力，而是**拜访量**和**关系建立**。

### 4️⃣ 陌拜工厂实战步骤
**第一步：直接冲工厂/门店**
> "你就直接去，不要管 3721，直接就去冲门店或者工厂。"

**第二步：参观设备**
- 看机器品牌和使用年限
- 询问故障率和更换频率
- 数工人数量，估算规模

**第三步：从工人获取信息**
> "一人一包烟和一瓶红牛，从工人那里可以问到竞品、囤货量、管理层情况等。"

**第四步：见负责人/老板**
- 提前预约
- 准备自我介绍（手写或对着镜子练）
- 准备资料

### 5️⃣ 聊天话术设计
**铺垫话术：**
> "你这个设备哪一年生产？使用了几年？进货价是多少？"

**引导话术：**
> "你们产能这么厉害，认识这边头部吧？"（拍马屁）

**展示专业：**
> "我去过其他同类型工厂，他们的生产模式是...经营模式是..."（没去过也要吹得像真的）

**提出需求：**
> "现在是否需要更换一些东西？有没有这个想法？"

### 6️⃣ 一找二策略（核心）
> "去搞到这个客户，一般来说都是一找二，绝对不是弯弯弯这个逻辑。"

**操作步骤：**
1. 邀请现有客户去第三家工厂考察
2. 介绍同行更先进的层面
3. 一来二去就能吃上一顿饭
4. 加微信，发位置
5. 一周后再次拜访
6. 聊地理人文、所见所闻、其他客户的故事
7. 约吃饭
8. 再过一周聊具体安排、预算、定金、合同细节

**关键：** 交叉性质获得人脉，快速建立信任。

### 7️⃣ 成交流程
```
第一次拜访：建立联系，了解情况
一周后第二次：聊天熟络，挖掘需求
再过一周第三次：聊具体安排，确认预算
谈好后：签合同，发货
```

**注意：** 新人需要老大出面帮忙谈，争取更多利益空间。

### 8️⃣ 跟进客户的逻辑
**目的分类：**
- **老客户**：靠逼单（完成业绩指标）
- **新客户**：靠下单（开发潜力客户）
- **新开发客户**：靠开单（建立信心）

**业绩分配比例示例：**
- 大客户：80%
- 中客户：15%
- 小客户：5%

### 9️⃣ 畅销 + 滞销搭配策略
> "我做产品的时候，一直讲究的是畅销搭配。好卖的，我往往要去搭一些不好卖的。"

**案例：** 卖中华配杂烟，卖茅台配杂酒

**目的：**
- 避免库存积压
- 完成公司单品 SKU 出货量考核
- 清库存，减少资金压力

### 🔟 逼单技巧
**对老客户：**
- 提前做下个月促销活动
- 跟小客户或销售渠道沟通，做好让利计划
- 囤积最畅销的品
- 风险降到最低

**对新客户：**
- 给最畅销的产品
- 每个品类挑一个最好卖的
- 提前梳理下游市场
- 确保上架后能动销

**警示：**
> "不能贪心，不能因为业绩压力就一下子给太多，消化不了库存，潜力客户就被你撑死了。"

### 1️⃣1️⃣ 维护客户的七步法

**第一步：了解市场动态**
- 走访市场，检查客户做得行不行
- 提供改进和弊端方案
- 拉进关系，展示专业

**第二步：从公司争取资源**
> "你必须要学会从公司里面多争取资源。"

**争取内容：**
- 机制有没有？
- 返利有没有？
- 折扣有没有？
- 赠品有没有？
- 物资有没有？

**第三步：陪同走访市场**
- 跟客户一起做方案
- 实地了解市场情况

**第四步：培训激励客户人员**
- 提升客户团队专业性
- 提供奖励品（公司出钱）
- 加深感情

**第五步：经验交流**
- 同行私下交流
- 地方性资源对接
- 人情世故

**第六步：中立立场**
> "业绩是公司给的，饭碗是客户给的。怎么维持绝对中立，是立足之道。"

**第七步：长期主义**
- 每天列任务，完成客户拜访
- 该打的电话一个都不能少
- 积累几个核心大客户/代理

### 1️⃣2️⃣ 需求挖掘三种情况

**情况一：客户满足现状，需创造需求**
**方法：**
- 透露竞争对手正在上新系统
- 告诉客户同行用了产品后成本降低 30%
- 制造危机感

**情况二：客户有需求但打官腔**
**方法：**
- 多角色验证
- 找甲方内部其他人
- 找客户的其他供应商
- 找竞争对手
- 多渠道验证，方向性就解决了

**情况三：客户信息删减或改动**
**识别方法：**
> "当接触到一定程度时，就问：如果我们达到了什么条件，是不是就可以当天签合同？合同多少天能签下来？"

**目的：** 通过封闭性问题判断对方态度，排除白嫖客户。

### 1️⃣3️⃣ 销售认知框架

**零售类销售：**
- 产品卖给谁 → 客户分析
- 在哪里卖 → 产品定位
- 怎么卖更快 → 销售方式
- 促销活动 → 写在简历里

**渠道类销售：**
- 适合哪些渠道 → KA 商超/便利/特通/线上
- 渠道怎么拓展 → 新店开发
- 怎么维护 → 新老客维护/产品维护/单品维护

**大客户销售（销冠思维）：**
- 决策者是谁？
- 预谈判方案有没有？
- 想达到什么目的？
- 围绕哪些点打动客户？
- 公司/产品/自己的优劣点是什么？
- 如何化解劣势？
- 谈判技巧：怎么认识、怎么谈、谈什么、兴趣点、顾虑、破除顾虑

### 1️⃣4️⃣ 销售逆袭机会
> "销售这个行业，是很多人逆袭的一个很好的机会。"

**必备能力：**
1. **产品知识**：研究从研发到生产到售前售后全流程
2. **个人 IP**：建立个人形象，提高综合能力
3. **知识储备**：建立自己的起手势和销冠方法
4. **主动学习**：越底线越高、行业越高，涉及知识面越杂

### 1️⃣5️⃣ 销售像一盘生意
> "等你未来有一天，你会发现销售的很多活动，它就是像一盘你自己的生意。"

**关键：**
- 怎么布局是关键
- 平时不能懒
- 该跑的客户、代理、合作伙伴每天都要跑完
- 该打的电话一个都要打
- 有了几个核心大客户后，就是长期主义

## ⚠️ 隐藏假设与风险

### 行业适用性
- **假设**：工厂/门店陌拜方法适用于所有 2B 行业
- **现实**：某些行业（如 SaaS、金融）可能不适用
- **建议**：提取底层逻辑，结合行业特点调整

### 个人风格依赖
- **假设**：一人一包烟一瓶红牛的方法可复制
- **现实**：不是每个人都擅长社交、喝酒
- **建议**：找到适合自己的关系建立方式

### 时间投入
- **假设**：每天拜访客户、打电话可执行
- **现实**：新人可能需要兼顾学习产品知识
- **建议**：合理分配时间，前期多拜访，后期多学习

### 公司资源限制
- **假设**：能从公司争取到返利/折扣/赠品
- **现实**：小公司可能没有这些资源
- **建议**：用专业和服务弥补资源不足

## 🚀 实践应用建议

### 对于销售新人（0-3 个月）
1. **找师傅**：尽快找到一个愿意带你的大师傅
2. **陌拜**：每天至少拜访 3-5 个客户
3. **开第一单**：三个月内必须开出第一单，保命
4. **学习产品**：研究从研发到售后全流程
5. **建立话术**：准备自我介绍、产品介绍、案例分享

### 对于 2B 销售（3 个月 -2 年）
1. **一找二策略**：邀请客户考察，建立深度关系
2. **争取资源**：向公司申请机制/返利/折扣/赠品
3. **客户维护**：七步法，定期拜访，提供价值
4. **畅销搭配**：好卖的搭不好卖的，完成考核
5. **培训客户**：提升客户团队专业性

### 对于资深销售（2 年+）
1. **长期主义**：积累几个核心大客户/代理
2. **布局思维**：把销售当自己的生意来经营
3. **中立立场**：业绩是公司给的，饭碗是客户给的
4. **经验传承**：带新人，建立团队
5. **资源整合**：同行交流，人脉互换

## 🔑 核心洞察

### 1. 销售的本质
**不是技战术，不是学习能力，而是：**
- 拜访量
- 关系建立
- 信任积累

### 2. 第一单的意义
**不只是业绩，更是：**
- 证明自己可以卖货
- 度过试用期，保住薪资
- 获得领导信任，减轻压力
- 建立信心，进入正循环

### 3. 客户维护的底层逻辑
**七步法的核心：**
- 提供价值（市场动态、改进方案）
- 争取资源（返利、折扣、赠品）
- 培训团队（提升客户能力）
- 中立立场（业绩是公司给的，饭碗是客户给的）

### 4. 销售的终极形态
> "销售像一盘你自己的生意。"

**从打工者心态到老板心态：**
- 布局思维
- 长期主义
- 资源整合
- 团队建立
